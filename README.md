# Subject: Software technology
Demo Books App With Jetpack Compose and MVVM Architecture.

## Setup Project

* **Require**
    + Environment: JDK – Java Development Kit và Android SDK.
    + IDE Android Studio.
   
* **clone & run project**
  
    + **Step 1:** Coppy link: 
      ```sh
      https://github.com/HuynhTai02/CNPM_JetpackCompose.git
      ```
    + **Step 2:** Open Android Studio.
      
    + **Step 3:** Install project from Github to Android Studio:
      + Choose **Get From VCS** on screen **Welcome to Android Studio**.
      + Choose **Git** in the list of version control systems **(Version control)**.
      + Paste the copied link into the field **URL**.
      + Choose folder to be saved project to field **Directory**.
      + Click **Clone** to start the download process project from Github.

    + **Step 4:** **Wait** download and configuration process project **completed**:
      + Android Studio to be installed project from Github, and automatically configure the project.
          
    + **Step 5:** Android device configuration
      + Use **virtual machine ADV** to run the application.
      + Or connect with **device Android** your. Make sure your Android device is turned on **developer** and **Debug USB** or **Debug Wifi** has been activated.
          
    + **Step 6:** Run project:
      + Click button **Run > Run 'app'** in Android Studio to run the project..
      + Wait for Android Studio to compile and install the app on your Android device or virtual machine.
      + Once the compilation and installation process is complete, the application will be launched on your device or virtual machine.

    + **Step 7:** **Experience** project **Books app**.
